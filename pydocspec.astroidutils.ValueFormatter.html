class ValueFormatter: (source)

View In Hierarchy

Formats values stored in AST expressions back to source code. Used for presenting default values of parameters and annotations.

Note
The default behaviour defers to astroid.nodes.NodeNG.as_string. This should be overriden if you want more formatting functions, like outputing HTML tags.
