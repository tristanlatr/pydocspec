pydocspec.loader

module documentation
(source)

Our own version of the docspec loader.
Note
The current implementation is largely adapted from pydoctor's AST builder, simply based on the ast module. Because of that, it is very fast. But single line comments (starting by "#") are ignored. Except for type comments, that are supported by the AST module.
